NEXORA DESKTOP – ÜBERSETZUNGSQUELLDATEI
=========================================

ANLEITUNG:
1. Kopieren Sie den gesamten folgenden Inhalt (Strg+A, Strg+C).
2. Gehen Sie zu https://translate.google.com.
3. Wählen Sie: Englisch → [Zielsprache].
4. Fügen Sie den Text in das linke Feld ein (Strg+V).
5. Kopieren Sie das übersetzte Ergebnis aus dem rechten Feld.
6. Speichern Sie die Datei unter: es_translation.txt, fr_translation.txt oder de_translation.txt

REGELN:
- Übersetzen Sie niemals Text innerhalb von eckigen Klammern.
- Ändern Sie niemals {{placeholder}} wie {{count}}, {{total}}, {{used}}, {{name}}, {{version}}, {{msg}}, {{date}}, {{step}}, {{message}}, {{cores}}, {{threads}}.
- Technische Begriffe wie VMAF, LUFS, FFmpeg, FFprobe, GPU, NVENC, AMF, QSV, CPU, RAM, H.264, H.265, HEVC, ProRes, DNxHD, AV1, AAC, PCM, MOV, MP4, MXF, TS, SHA-256, dBTP, fps, kbps, Mbps, GB, MB, Node.js, SQLite, Tauri, React, Rust, Nexora, BBC, Netflix, DPP, Windows, macOS, Linux, VS Code, npm, LUFS, LU, EBU R128, R128 dürfen NIEMALS übersetzt werden.

– Zeichensetzung und Formatierung beibehalten.
– Pfeilsymbole wie ← beibehalten. Wie besehen
